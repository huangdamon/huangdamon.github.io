类型

布尔类型

关键字bool,不支持自动或者强制的类型转换

整型

  1. int 和int32是两种不一样的类型，因为不会做自动转换，可以强制转换。
  2. 不同类型的整形数不能直接比较，但是可以和字面量比较。

float32 和 float64

浮点数比较可以使用math包

字符串

数组

数组是值类型，因为值类型变量在赋值和作为参数都是产生一次复制动作

数组切片

第三章 面向对象编程

类型系统

interface{} 是一个空接口

因为go语言类型都是基于值传递，所以想改变变量的值，只能传递指针

值语义和引用语义

基本类型，如 byte 、 int 、 bool 、 float32 、 float64 和 string 等；
复合类型，如数组（array）、结构体（struct）和指针（pointer）

结构体

匿名组合

采用组合的文法提供了继承，所以称之为匿名组合

需要注意是

  1. Base的方法中无法访问Foo的成员方法和变量
  1. 匿名组合类型相当于以其类型名称（去掉包名部分）作为成员变量的名字(这个编译异常不一定会出现，一旦调用者调用其中某个重复的定义，就会出现问题)

可见性

Go语言的可见性是大小写字母区分， Go语言中符号的可访问性是包一级的而不是类型一级的

接口

只要实现接口要求的所有函数，就等于实现了该接口

接口赋值
  1. 两个接口定义的方法相同
  2. 如果A的方法列表是B的方法列表的子集，那么接口B可以赋值给接口A
